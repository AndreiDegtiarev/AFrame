AFrame Documentation

Introduction

AFrame (arduino framework) library that implements basis functionality for ASensor and AWind libraries. In the library main events handlers and wrapper class ISensor are implemented. ISensor is a kind of plug-in interface for external sensor libraries

Documentation

API documentation: http://andreidegtiarev.github.io/AFrame/

Installation

To install you need download https://github.com/AndreiDegtiarev/AFrame/archive/master.zip and unzip it into arduinosketchfolder/libraries/ folder and then remove the "-master" in the folder names.
